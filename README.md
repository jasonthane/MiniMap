MiniMap is a small personal navigation device based on the ESP32-S3, a GNSS receiver, a TFT display and an optional battery.

## Hardware

The initial implementation uses the Adafruit ESP32-S3 Reverse TFT Feather, featuring a ST7789 TFT display (240×135), 4MB Flash and 2MB PSRAM:

https://www.adafruit.com/product/5691

along with the Adafruit Ultimate GPS FeatherWing based on the MTK3333 chipset:

https://www.adafruit.com/product/3133

## Getting Started

Just solder these two boards together using the included header pins, plug a USB-C cord into the Feather, fire up the Arduino editor, and you're in business!
